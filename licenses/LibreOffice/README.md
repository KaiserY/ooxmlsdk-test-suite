# LibreOffice Corpus

Source: https://github.com/LibreOffice/core

The corpus files are kept under `corpus/LibreOffice/` with their original
relative paths from the source project.

This directory contains license files copied from LibreOffice core. The copied
corpus files retain their upstream licensing terms.

The binary Office corpus was synchronized from LibreOffice core commit
`9feef7299e94a9f68f0e715700b6efc06073c78d`. The checked-in `COPYING`,
`COPYING.LGPL`, and `COPYING.MPL` files match that commit.
